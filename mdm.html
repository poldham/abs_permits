Master Data Management

Paul Oldham

29/07/2018

This document aims to support Master Data Management planning by identifying the main data sources for development of a Monitoring System for Access and Benefit Sharing. The table below can be downloaded as a csv file here.

category source type url r python description
geographic GBIF api https://www.gbif.org/developer/maps NA NA Mapping API for taxonomic data from GBIF
geographic geonames text http://download.geonames.org/export/dump/allCountries.zip NA NA 11 million georeferences place names.Field headings in main page read me
geographic google api https://cloud.google.com/maps-platform/ NA NA maps and places services
literature crossref api https://github.com/CrossRef/rest-api-doc rcrossref habanero literature metadata, restful, json, R and python packages, solr search syntax
literature core.ac.uk api https://core.ac.uk/docs/#!/all/search rcoreoa pyoacore full texts of scientific literature
literature unpaywall api https://unpaywall.org/products/api roadoi NA identify open access publications in crossref
literature Biodiversity Heritage Library api https://biodivlib.wikispaces.com/Developer+Tools+and+API rbhl NA Historic biodiversity publications
literature Open Academic Graph text https://aminer.org/open-academic-graph NA NA metadata for 330 million scientific publications
patent patentsview text http://www.patentsview.org/download/ NA NA US patent data in table format
patent patentsview api http://www.patentsview.org/api/patent.html NA NA US patents end point. Limited text search, lookup for inventors or applicants
patent USPTO xml http://patents.reedtech.com/parbft.php NA NA US patent applications full text
patent USPTO xml http://patents.reedtech.com/pgrbft.php NA NA US granted full text
